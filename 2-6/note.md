i++: 后自增，自增后的值不会赋给i，仅仅是一个副本。
++i: 先自增, 自增后的值会赋值给i。

```c
int a;
        a = ((a=3*5, a*4), a+5);
```
结果是：20
```c
int a;
        a = (a=3*5, a*4), a+5;
```
结果是：60

误区：以为是a=3*5, 然后 a = a*4
其实是 a = 3*5 = 15, 然后倒数第一个逗号的左边变为a*4 = 60, 即 `a = 60, a+5，`。然后进行操作 a = 60，并返回a+5。所以a还是60，没有变量接受 a+5，所以 a+5 毫无意义。

而对于第一个程序，等效于 `a = 3*5; a = (60, a+5)`，所以最后结果是 20。
